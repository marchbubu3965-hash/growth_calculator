# 👶 寶寶生長管理系統 (Baby Growth Tracker)

這是一個專為家長設計的網頁工具，用於記錄並推算寶寶的身高與體重發展。

### 🚀 核心功能
* **0-12 歲精準模擬**：採用分段階梯式生長模型，避免線性預測的巨大誤差。
* **今日數據推估**：根據歷史紀錄，自動推算出寶寶「今天」理論上應有的身高體重。
* **資料備份**：支援匯出成 Excel (CSV) 格式，並提供 CSV 匯入功能。
* **本地儲存**：所有資料儲存於您的瀏覽器中，保障個人隱私。

### 📊 生長模型設定 (0-12 歲)
本系統將成長階段細分為：
1. **0-1歲**：極速成長期 (+2.0 cm/月)
2. **1-2歲**：快速成長期 (+1.0 cm/月)
3. **2-4歲**：穩定期 (+0.6 cm/月)
4. **4-9歲**：平穩期 (+0.45 cm/月)
5. **9-12歲**：青春期前期衝刺 (+0.55 cm/月)
*系統預設最高計算至 12 歲。*
